What is Spec ADE?

Understanding Spec ADE

01

1. What is Spec ADE?

Spec ADE is a next-generation desktop workspace designed from the ground up for AI coding agents. Instead of inserting AI into a traditional text editor, it wraps multiple terminals, browsers, databases, and Git into a single binary that agents can control natively.

02

2. What is it for?

It automates entire developer workflows—like implementing features, refactoring databases, fixing bugs, and running test suites. Rather than manually writing every line of code, you direct autonomous agent pools (like Claude Code, Auggie, or Codex) to execute complex engineering goals.

03

3. How does it work?

First, describe what you want (Goal). Next, Spec ADE generates a structured task list (Spec). The agents execute local file edits, terminal runs, and SQL queries in parallel. Finally, you review the progress and approve changes at key checkpoints.

Architecture

Built for performance, designed for trust

Spec ADE is a Rust + Vue 3 stack that ships as a single binary. Frontend and backend speak REST, WebSocket, and SSE; everything else stays local.

UI Vue 3 + Quasar v2 + Pinia

Frontend (Vue 3 + Quasar + Pinia)

Fifty-plus Pinia stores in Composition API, recursive pane tree, xterm Teleport pool, CodeMirror 6 with Vim and minimap, and a context-aware shortcut store.

API Rust 2024 · Axum · tokio

Backend (Rust 2024 + Axum)

PTY manager, ACP threading model, hybrid Git engine, six-driver DB layer, Goal/Claw runners, license verification — all coordinated through a typed AppState.

DATA settings.json · OS keychain

Storage

Single `~/.config/spec-ade/settings.json` file. Passwords go to the OS keychain (with AES-GCM headless fallback). License keys come from the Ed25519-signing portal.

Protocols

  • WebSocket PTY I/O, ACP relay, DB streaming

  • Proxy /fw/{port}/{*rest} forwards HTTP + WS

FAQ

Frequently asked questions

Honest answers about access, privacy, and how Spec ADE works.

01 Is Spec ADE open source?

Not yet — Spec ADE is a private product developed by the viber.vn team. The source code is not public today. We plan to open-source it in the future once the product reaches stable maturity.

02 How much does it cost?

Spec ADE is free to use during the current private beta. Pricing for future tiers will be announced separately — beta users will get advance notice and grandfathered terms.

03 How do I install it?

Run `npx -y @spec-ade/cli@latest` in your terminal. The CLI bootstraps Spec ADE on your local machine or your own VPS and opens it in your browser automatically. No need to clone a repo or compile anything yourself.
